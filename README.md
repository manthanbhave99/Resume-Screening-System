## 📌 Overview

An AI-powered Resume Screening System that automates candidate evaluation using NLP-based skill extraction and scoring.

---

## ✨ Key Features

* 🔐 **Role-Based Authentication**

  * Admin Login
  * User Login
  * Signup System

* 📂 **Resume Parsing**

  * Supports PDF, DOCX, TXT
  * Extracts text automatically

* 🧠 **AI Screening Engine**

  * Skill extraction
  * Resume-job match scoring

* 📊 **Admin Dashboard**

  * View registered users

* 🔄 **Session Management**

  * Login / Logout functionality

---

## 🛠️ Tech Stack

* **Frontend:** Streamlit
* **Backend:** Python
* **Database:** SQLite
* **Processing:** NLP-based parsing

```bash

```
